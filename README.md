# 🧮 Python Calculator

A simple calculator with graphical user interface (GUI) built with Python and Tkinter.

## ✨ Features

- ➕ Addition
- ➖ Subtraction
- ✖ Multiplication
- ➗ Division
- 📋 Calculation History
- 🔄 Continue with previous result
- ❌ Divide by zero protection
- 🧹 Clear screen

## 🚀 How to Run

### Prerequisites
- Python 3.x installed

### Run the program
```bash
python calculator.py
```

Build executable (optional)

```bash
pip install pyinstaller
pyinstaller --onefile --windowed calculator.py
```
